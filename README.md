A Chrome extension that acts as your personal WhatsApp demon, handling tasks while you watch from another window.

## Features

- Press play and watch the demon take control
- Sends predefined messages to contacts
- Works across multiple windows
- Real-time visual feedback of actions
- Dark mode interface with demon animations

## Installation

1. Clone repository:
```bash
git clone https://github.com/yourusername/whatsapp-assistant-demon.git
cd whatsapp-assistant-demon
```

2. Load extension in Chrome:
   - Open `chrome://extensions/`
   - Enable Developer Mode
   - Click "Load unpacked"
   - Select the extension directory

## Usage

1. Click extension icon
2. Press "Start Demon" button
3. Watch demon execute tasks in WhatsApp Web
4. Monitor progress in extension popup
